# installerwindows.fr

Front-end installerwindows.fr

-   Next.JS@13
-   React@18.2
-   Typescript@4.8.4
-   SASS@1.55

# install

Pour installer les packages :

```
npm install
```

# dev

Pour démarrer le front en environnement de dév

```
npm run dev
```

# build

Pour démarrer le front en environnement de prod

```
npm run build
npm start
```
